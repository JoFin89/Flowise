# EnnéaPocket

Prototype PWA mobile-first pour l’ennéagramme (FR), construit avec Next.js App Router, TypeScript et Tailwind.

## Scripts

```bash
pnpm dev
pnpm build
pnpm start
pnpm test
```

> Remplacez `pnpm` par votre gestionnaire préféré (npm/yarn). Le package vit en workspace et hérite des dépendances du monorepo.
